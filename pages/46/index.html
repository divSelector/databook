Header 2. Sidebar and Main Content (wrap these together in a div with display: flex) The Sidebar has a fixed width, while Main content has 100% width. All of this is wrapped in a container div that has a max-height and max-width (you decide what size you want) and position: absolute (unless on mobile, where i set it back to position: static using media queries so it takes the full screen). Here's how it's set up for this website: #websiteContainer { max-height: 928px; max-width: 700px; top: 0; bottom: 0; left: 0; right: 0; margin: auto; position: absolute; } This removes position: absolute for mobile: @media (max-width: 641px) { position: static; } I have multiple widths set up for different window sizes using media queries. You can also skip the position: absolute and max height if you want your page to be higher and not centered. If you have more questions let me know.
